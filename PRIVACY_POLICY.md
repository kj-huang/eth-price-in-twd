# Privacy Policy — PriceScope

**Last updated:** 2026-03-20

## Data Collection

PriceScope does **not** collect, store, or transmit any personal data. The extension operates entirely within your browser.

## Network Requests

The extension makes requests to the following third-party APIs solely to retrieve price data:

- **Binance API** (`api.binance.com`) — cryptocurrency prices
- **Yahoo Finance** (`query1.finance.yahoo.com`) — stock prices
- **ExchangeRate API** (`open.er-api.com`) — currency exchange rates

No user-identifiable information is included in these requests.

## Local Storage

Price data is cached locally using Chrome's `storage.local` API to enable the popup display. This data stays on your device and is never shared.

## Permissions

- `alarms` — schedule periodic price refreshes
- `storage` — cache price data locally

## Contact

For questions about this privacy policy, open an issue on the GitHub repository.
